小程序构建npm失败

小程序中构建npm不能成功创建，踩坑后进行记录，避免下次又掉坑里

小程序构建 npm 失败

官方文档地址(官方是错误的)
csdn 参考地址
小程序从基础库版本 2.2.1 以上就开始支持 npm 安装第三方包了，看了文档之后
首先，如果你按文档走下去：
1、执行命令安装 npm install
2、工具 -> 构建 npm
恭喜你，将会看到没有找到 node_modules 目录。
问题原因：
一直开发小程序，没怎么用过 npm 了，特么的发现 npm 还没有初始化，就直接让人安装了（文档太笼统）,还有就是 npm install 加上安装的包名
正确步骤：
1、在小程序根目录下执行命令：
npm init
2、在继续安装你要安装的第三方包：
npm i vant-weapp -S –production
3、开发工具：工具 -> 构建 npm
4、详情里面：选中使用 npm 模块
