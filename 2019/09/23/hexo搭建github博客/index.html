hexo搭建github博客

本文介绍了怎么使用GitHub和hexo进行博客的搭建以及部署

Hexo 搭建博客

步骤:

  1. 确保电脑安装了 node 环境和 git 环境
  2. 创建 git 仓库
    • 具体参考: 下方知乎的回答中的安装 Git 一节
  3. npm install hexo-cli -g 全局安装 hexo 脚手架
  4. 安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。
  5. hexo init
  6. 安装依赖 npm i
  7. npm install hexo-deployer-git –save
  8. 在 _config.yml 中配置 git 仓库
    deploy:
      type: git
      repo: https://github.com/Marla62/Json.github.io.git #git 仓库地址
      branch: master
  9. 常用命令:
    hexo n “我的博客” == hexo new “我的博客” #新建文章
    hexo g == hexo generate #生成
    hexo s == hexo server #启动服务预览
    hexo d == hexo deploy #部署
  10. 引入评论插件 Gittalk
